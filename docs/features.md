# MongoFlow Features

## Core workflow
- Browser-based MongoDB interaction
- Sample-based schema inference
- Type-aware controls
- Visual aggregation construction
- Multi-language code export
- Post-execution statistics
- AI-assisted query explanation and correction

## Supporting database-client functions
- Connection management
- Document browsing and editing
- Integrated MongoDB shell
- Index management
- Validation-rule management
- Query history
- Breadcrumb navigation

## Current limitations
- No formal WCAG audit yet
- No comprehensive automated test suite yet
- AI output is not formally verified
- AI use may send database context to the configured provider
- Not production-ready as a public multi-tenant service without additional security mechanisms
