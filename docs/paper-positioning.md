# Paper Positioning

MongoFlow is submitted as an original software publication to SoftwareX. The paper therefore follows the SoftwareX article template rather than the structure of a conventional research article.

The contribution of MongoFlow is not a new database algorithm or a new AI model. The contribution is a reproducible open-source software artifact that integrates:

- Web-based MongoDB interaction
- Schema-aware exploration
- Visual aggregation pipeline construction
- Polyglot code generation
- Execution feedback
- AI-assisted query generation and correction

The manuscript discusses background, related tools, and contribution inside the “Motivation and significance” section because this follows the SoftwareX template structure.
