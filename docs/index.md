# MongoFlow Documentation

This folder contains technical and research-oriented documentation for MongoFlow.

MongoFlow is an open-source, web-based MongoDB IDE for document exploration, sample-based schema inference, visual aggregation construction, multi-language code export, post-execution statistics, and optional AI-assisted query explanation and correction.

## Documentation Files

- `features.md`: overview of core and supporting features
- `architecture-rationale.md`: architecture design, alternatives, and trade-offs
- `query-representation.md`: internal aggregation-pipeline representation
- `ai-assistant-design.md`: AI prompt construction, context, limitations, and safeguards
- `security.md`: security assumptions and deployment considerations
- `limitations.md`: technical, evaluation, AI, and deployment limitations
- `accessibility.md`: current accessibility status and planned checks
- `testing.md`: current testing status and planned automated tests
- `related-tools-comparison.md`: comparison criteria and positioning against existing tools
- `paper-positioning.md`: positioning of MongoFlow as a SoftwareX software paper
- `research-use.md`: possible research uses of MongoFlow
- `softwarex-metadata.md`: metadata used for the SoftwareX submission
