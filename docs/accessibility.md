# Accessibility Status

MongoFlow has not yet undergone a formal WCAG accessibility audit.

The interface uses reusable UI components and responsive styling, but formal accessibility compliance has not been verified.

## Current Status

The current version has not been systematically evaluated for:

- keyboard navigation;
- focus order;
- screen-reader labels;
- color contrast;
- form-label clarity;
- error-message accessibility;
- ARIA attributes;
- accessible table and tree navigation.

## Planned Accessibility Checks

Future accessibility work should include:

1. Keyboard-only navigation testing.
2. Screen-reader testing.
3. Color-contrast analysis.
4. Focus-order verification.
5. Form-label review.
6. Accessible error-message review.
7. Accessibility testing of the schema tree and aggregation builder.
8. Evaluation of generated-code blocks for readability and navigation.
